## About project
### 독서 모임을 만들고 같이 책을 읽어요
- 📚 Tab1 : 독서 모임
  - 독서 모임을 만들고 모임에 참여할 수 있어요
  - 같이 책을 읽으며 책에 대한 의견을 나눠요
- ⏲️ Tab2 : 독서 타이머
  - 여러분의 독서 시간과 페이지를 기록할 수 있어요
  - 같은 책을 읽고 있는 사람들과 같이 타이머를 공유할 수 있어요
